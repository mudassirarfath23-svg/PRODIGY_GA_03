
# PRODIGY_GA_03

## Project Title
Text Generation with Markov Chains

## Overview
This project implements a text generation system using the Markov Chain algorithm in Python. The model learns patterns from a sample text dataset and generates coherent sentences based on user-provided topics.

## Features
- Generates text using Markov Chains
- Accepts user-defined topics
- Produces AI-related sentences
- Simple and beginner-friendly implementation

## Technologies Used
- Python
- Markovify
- Google Colab

## Project Workflow
1. Install the Markovify library.
2. Create a text dataset.
3. Train the Markov Chain model.
4. Generate text based on a topic.
5. Display the generated output.

## Result
Successfully generated meaningful text related to Artificial Intelligence and Cloud Computing using a Markov Chain model.
